# aplikasi-klinik-sederhana-master

Aplikasi ini adalah aplikasi klinik sederhana yang dibangun menggunakan framework CodeIgniter.

## Persyaratan

- PHP 7.2 atau lebih baru
- MySQL atau MariaDB
- Composer
